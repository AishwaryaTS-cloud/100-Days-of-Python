## 🧪 Practice Code Snippets

```python
# Type checking
print(type(10))
print(type(3.14))
print(type("Aishwarya"))
print(type(True))

# Type conversion
num = "50"
num_int = int(num)
print(num_int + 10)

# Math operations
print(7 // 2)   # Floor division
print(7 % 2)    # Remainder
print(2 ** 3)   # Exponent

# Number manipulation
value = 3.4567
print(round(value, 2))

# f-string example
age = 20
print(f"I am {age} years old.")

```
